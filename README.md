# Visual SourceBook

Visual SourceBook is a reference manager that lets users save source page URLs, add titles and comma-separated tags, and optionally include thumbnails for quick visual reference.

## Features

- Save source page links as references
- Add titles and comma-separated tags
- Optional thumbnail image
- Search references by tags
- Data stored locally using browser localStorage

## How to Use

1. Click **Get Started** to open the workspace.
2. Use the **Add** button to save a new reference.
3. Add the source URL, title, tags, and an optional thumbnail.
4. Search references using tags.

## Tech Stack

- React
- Vite
- CSS Modules
- localStorage
